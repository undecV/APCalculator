## How to Use

1. Open [AP Calculator](https://undecv.github.io/APCalculator/).
2. Enter the values.
3. View the results.

## Features

- Supported languages: English, Traditional Chinese.
- Real-time interaction.
- Basic input validation.
- Supports JSON import and export.
- Single HTML file.

## Notes
